# 🍔 Food Delivery Website

A modern and responsive **Food Delivery Website** built using **HTML, CSS, and JavaScript**. The project demonstrates responsive web design, clean UI implementation, interactive user experience, and front-end development fundamentals.

Designed to provide an engaging browsing experience across desktop and mobile devices.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Harshaapoorv/Food-Delivery.git
```

### Navigate to the Project

```bash
cd Food-Delivery
```

### Run the Project

Simply open **index.html** in your preferred browser.

Or use the VS Code Live Server extension for a better development experience.
